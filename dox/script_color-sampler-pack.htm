Colors Sampler Pack : [full update] All the color schemes on vim.sf.net (now with organized menu)

 script karma  Rating 755/277, Downloaded by 12370

created by
Robert (MetaCosm)
 
script type
color scheme
 
description
This package is simply to help people who want to look at alot of color schemes.  It is all the color schemes on vim.sf.net as of March 13th, 2005 -- zipped up in a single package.  There are 140 of them included.  Every single theme was updated to its newest revision.

Now it includes a nice little menu plugin that makes it very easy to try out the different themes.  

Themes->Black->BillW
Themes->Grey_Brown->MetaCosm
...

This package was put together simply to save others time in downloading the colorschemes.  If you are a scheme author and for some reason do not want your theme included, please simply add a comment that notes such, and I will remove it.
 
install details
Simply unzip, and place the files in ~/.vim/plugin and ~/.vim/colors --- it will unzip with correct dir structure, so you can just unzip to ~/.vim

Hope this helps you... join #vim on irc.freenode.net for all your vim help needs :)
