uri-ref : Easily insert references to URLs in emails

 script karma  Rating 20/8, Downloaded by 278

created by
Hugo Haas
 
script type
utility
 
description
Inline URIs in emails make the text around them obscure. This is why I put references to URIs at the bottom of the email.

This script provides a prompt to do so in a painless way.

Examples and documentation are at: http://larve.net/people/hugo/2001/02/email-uri-refs/
 
install details
Just stick the script into your plugins directory or your vimrc.

While writing an email, in insert mode, press F5 and follow the instructions.

Note that your mail has to have a valid signature for the script to work as it's currently written.
