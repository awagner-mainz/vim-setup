latex-mik.vim : LaTeX support (MikTeX) for GVIM on Win32-systems

 script karma  Rating 31/15, Downloaded by 455

created by
Volker Kiefel
 
script type
utility
 
description
This script adds a menu for LaTeX-related functions for .tex, .bib-files

To process multiple file LaTeX-projects, enter the project name (e.g.
``myproject'') at the prompt, if ``myproject.tex'' is the main file.

Menu options address:

  (pdf)latex
  bibtex
  makeindex
  yap (dvi-viewer)
  dvips
  gsview32
  tth

Latex-mik supports entering commands and environments.
 
install details
For installation, please put the script file (latex-mik.vim) into the vim plugin-directory
