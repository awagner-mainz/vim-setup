HiCurLine : Highlights the current line

 script karma  Rating 20/13, Downloaded by 969

created by
Charles Campbell
 
script type
utility
 
description
This script has a number of maps that
intercept various motions and cause the
current line to be highlighted.  It is
a variant of <HiMtchBrkt.vim>; see
vimtip#177.

    \hcli : initiate HiCurLine operation
    \hcls : stop HiCurLine operation

(see vimtip#411 to have HiCurLine automatically
begin highlighting)

 
install details
1. Put into your .vim/plugin or
   _vimfiles\plugin directory (Unix/Windows)
2. \hcli (actually <Leader> hcli) will
   begin the highlighting.
